Open Design Kit

A living toolkit for designing
with remote collaborators.

view on Github

Bocoup's Design team maintains Open Design Kit as an open source tool. It is useful for anyone who is designing with people who are not co-located. The Kit includes methods from ideation to implementation and is meant to evolve with use.

To share your feedback or add an method, open an issue or pull request on our Github repository.
